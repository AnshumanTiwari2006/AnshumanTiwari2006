Right now I'm mostly living in PyTorch and LangChain, occasionally arguing with Azure, and slowly turning side projects into things other people can actually use.

### On the bench right now

- **Automated theorem proving** — generating and verifying Lean 4 proofs on PutnamBench, training Energy-Based Models on the results. Math + ML is a weird, fun combination.
- **PySliceKit** — a PyPI package for slicing and manipulating arrays/tensors the way I always wished NumPy did it natively.
- **LineSight** — a visualizer that makes regression *feel* intuitive — loss surfaces, gradient descent, L1/L2 shrinkage, all interactive instead of stuck in a textbook.
- **EduQuiz** — an adaptive quiz platform, because most learning tools still don't adapt to the learner.
